О проекте

🔗 Link Keeper — хранитель ссылок

Это учебный проект, созданный для демонстрации навыков DevOps.

🛠 Технологический стек:
  • Backend: Python 3.12 + Flask
  • База данных: PostgreSQL 15
  • Кэш: Redis
  • Веб-сервер: Nginx (reverse-proxy + статика)
  • Контейнеризация: Docker + Docker Compose (3 реплики Flask)
  • CI/CD: Gitverse (GitLab CI совместим)
  • Облако: Selectel VPS
📦 Возможности:
  • Регистрация и аутентификация пользователей
  • Добавление, редактирование, удаление ссылок
  • Тегирование ссылок
  • Поиск по названию, описанию и тегам
  • Публичные и приватные ссылки
  • Кэширование популярных ссылок в Redis
  • Балансировка нагрузки через Nginx
👨‍💻 Автор:

Проект создан в рамках обучения DevOps.
